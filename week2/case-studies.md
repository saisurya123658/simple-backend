# Case Study 1: AI API Wrapper

## Problem

Many applications can call an AI model, but they often return inconsistent responses and fail when the API is unavailable. I wanted to build a reliable backend service that produces structured JSON responses, handles errors gracefully, and records API usage.

## What I Did

I built a backend wrapper using Node.js and Express that connects to an AI API. I designed endpoints to validate requests, return schema-based JSON, handle failures with proper HTTP status codes, and log API usage for monitoring and debugging.

## Outcome

The project demonstrated my understanding of backend architecture, REST APIs, error handling, and AI integration. It provides a reusable foundation for building AI-powered applications.

# Case Study 2: Temple Management System

## Problem

## What I Did

## Outcome

The project streamlined the booking process and strengthened my experience in backend development, database design, and REST API development.
