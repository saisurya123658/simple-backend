
# JavaScript 101 - Backend Assignment

## Overview

This project demonstrates the concepts learned from the JavaScript 101 presentation.

## Technologies

- Node.js
- Express.js

## Features

- Express Server
- REST API
- JSON Responses
- Middleware
- GET Routes
- POST Route

## API Endpoints

### GET /

Returns welcome message.

### GET /api/message

Returns a sample JSON response.

### GET /api/student

Returns student information.

### POST /api/feedback

Accepts JSON data.

Example Body

```json
{
    "name":"Sai Surya",
    "feedback":"Great Session!"
}
```
